# 📄 Flask Blog

📌 Описание
Flask Blog — это веб-блог, где пользователи могут создавать, редактировать и удалять статьи
-------------------------------------------------------------------------------------------

🚀 Технологии
--------------
• Python 3
• Flask
• SQLite
• HTML/CSS

⚙️ Функции
-----------
• Регистрация и авторизация пользователей
• Создание постов
• Редактирование и удаление постов
• Хранение данных в базе данных
• Добавление данных в базу данных
• Работа с пользователями (сессии, вход/выход)
• Защита маршрутов (доступ только для авторизованных)

🛠️ Установка и запуск
----------------------
1. Клонируйте репозиторий
2. Создайте виртуальное окружение
3. Установите зависимости -> Flask Flask-Login
4. Запустите сервер
