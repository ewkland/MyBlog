# MyBlog
Название проекта > My Blog
Описание > Простой блок для общения с системой Log-in на фреймворке Flask

Библиотеки >  Flask, Jinga2, MarkupSafe, Werkzeug и тд.
Языки > HTML, CSS, Python
Язык проекта > Английский

Инструкции по запуску / использованию проекта:

Запуск осуществляется через main.py
